# CEO Productivity Dashboard

生産性の高い社長の1日を管理する包括的なダッシュボードアプリケーション。

## 機能

### 🛌 睡眠管理
- 睡眠時間の記録（就寝時間・起床時間）
- 睡眠品質のパーセント表示
- 睡眠段階の分析（深い睡眠・浅い睡眠・REM睡眠）
- 週間睡眠トレンドの可視化

### 🍽️ 食事管理
- 食事名の入力と記録
- カメラでの食事写真撮影機能
- 自動栄養分析（カロリー、炭水化物、タンパク質、脂質、食物繊維、カルシウム、水分）
- AI栄養アドバイザーによる個別アドバイス
- 今日の食事一覧表示

### 🏃‍♂️ 運動管理
- 歩数の記録
- 消費カロリーの追跡
- 移動距離の計測
- 運動時間の記録
- 目標達成率の表示
- 週間運動トレンドの可視化

### ✅ タスク管理
- タスクの追加と優先度設定
- カンバン方式のタスク管理（未着手・進行中・完了）
- タスクのドラッグ＆ドロップ移動
- 生産性の可視化

### 📰 AIニュース
- 最新のAI関連ニュースの自動収集
- ビジネス戦略に関する情報提供

### 📊 データログ
- 日・週・月別のデータ表示
- 睡眠、栄養、運動、生産性のログ記録
- 視認性の高いデータ表示

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: Tailwind CSSによるレスポンシブデザイン
- **JavaScript**: ES6+によるモダンな実装
- **Chart.js**: データ可視化
- **Font Awesome**: アイコン表示

## 使い方

1. アプリケーションを開く
2. 各タブをクリックして機能にアクセス
3. データを入力して記録
4. ダッシュボードで全体の状況を確認

## 特徴

- **視認性の最適化**: グラデーション背景とガラスエフェクトによる美しいUI
- **レスポンシブデザイン**: あらゆるデバイスで最適な表示
- **リアルタイム更新**: データ入力後即座に反映
- **ローカルストレージ**: データの永続化
- **AIアドバイス**: 栄養バランスに関する個別提案

## インストール

1. リポジトリをクローン
```bash
git clone https://github.com/tls-client/memo1.git
```

2. プロジェクトディレクトリに移動
```bash
cd memo1
```

3. ブラウザで `index.html` を開く

## デプロイ

このアプリケーションは静的ファイルで構成されているため、任意のWebサーバーでホストできます。

```bash
# Pythonでローカルサーバーを起動
python -m http.server 8080

# Node.jsの場合
npx serve .

# PHPの場合
php -S localhost:8080
```
