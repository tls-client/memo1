# オクシモロンのポートフォリオ

## 概要

これはオクシモロンのポートフォリオウェブサイトです。Web開発とリバースエンジニアリングのスキルを紹介しています。

## 機能

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **ダークテーマ**: 目に優しいターミナルスタイルのデザイン
- **日本語対応**: 完全に日本語化されたインターフェース
- **インタラクティブなナビゲーション**: スムーズなページ遷移
- **スキル可視化**: アニメーション付きのスキルバー
- **プロジェクト展示**: 制作実績の詳細な紹介
- **お問い合わせフォーム**: 連絡先情報とコンタクトフォーム

## ページ構成

### メインページ (index.html)
- ホームセクション: プロフィール紹介
- 制作実績: 開発したプロジェクト一覧
- 自己紹介: 詳細なプロフィールとスキル
- お問い合わせ: 連絡先とフォーム

### ニュースツール (news.html)
- 最新ニュースの収集と表示
- カテゴリー別フィルタリング
- リアルタイム更新機能

## 技術スタック

- **HTML5**: セマンティックマークアップ
- **CSS3**: モダンなスタイリングとアニメーション
- **JavaScript**: インタラクティブ機能
- **Font Awesome**: アイコンライブラリ
- **レスポンシブデザイン**: CSS GridとFlexbox

## デザイン特徴

- **ターミナルスタイル**: ダークテーマにグリーンアクセント
- **モノスペースフォント**: プログラマー向けのデザイン
- **グラデーション効果**: 現代的なビジュアル
- **ホバーアニメーション**: インタラクティブな体験
- **スムーズスクロール**: ナビゲーションの最適化

## セットアップ

1. リポジトリをクローン:
```bash
git clone [repository-url]
```

2. ファイルをサーバーにアップロード

3. ブラウザで `index.html` を開く

## カスタマイズ

### プロフィール情報の更新
`index.html` 内の以下のセクションを編集:
- プロフィール名とタイトル
- SNSリンク
- スキルレベル
- 経験・実績

### プロジェクトの追加
`projects-grid` セクションに新しい `project-card` を追加:

```html
<div class="project-card">
    <div class="project-icon">
        <i class="fas fa-icon"></i>
    </div>
    <h3 class="project-title">プロジェクト名</h3>
    <p class="project-description">説明</p>
    <div class="project-tech">
        <span class="tech-tag">技術</span>
    </div>
    <div class="project-links">
        <a href="#" class="project-link">リンク</a>
    </div>
</div>
```

### カラーテーマの変更
`styles.css` のCSS変数を編集:

```css
:root {
    --accent-primary: #00ff88;  /* メインアクセント色 */
    --accent-secondary: #00ccff; /* サブアクセント色 */
    --bg-primary: #000000;     /* 背景色 */
}
```
